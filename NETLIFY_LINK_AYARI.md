# Netlify Link Ayarı

VSCode Run and Debug kısmında ilk seçenek artık Netlify adresini açar:

https://syncora.netlify.app/

Panel linki paylaşırken kullanman gereken adres:

https://syncora.netlify.app/

Dashboard/panel içi direkt adres:

https://syncora.netlify.app/app

Discord OAuth callback adresi:

https://syncora.netlify.app/discord/callback

Not: Netlify, bilgisayarındaki XAMPP MySQL'e bağlanamaz. Netlify ortamında `SYNCORA_STORAGE=netlify-blobs` kullan veya harici online MySQL bağla.
